Опис на проектот
GameBrain AI е интелигентен chatbot кој им помага на гејмери со совети, објаснувања и информации за видеоигри како Minecraft и Fortnite. Корисникот поставува прашање преку веб-интерфејс, а системот враќа јасен и разбирлив одговор. Проектот е наменет за гејмери кои сакаат побрзо да ги научат механиките, стратегиите и основните правила на игрите. Асистентот користи вештачка интелигенција преку Google Gemini API за да симулира разговор со корисникот. Проектот е корисен бидејќи овозможува брза помош без потреба од долго пребарување низ различни веб-страници. Дополнително, проектот покажува практична и реална примена на AI технологии.

AI API(и)
Google Gemini API – Се користи за разбирање на прашањата од корисникот и за генерирање текстуални одговори со совети, објаснувања и tips & tricks поврзани со видеоигри. Gemini API е избран поради бесплатниот пристап и одличните перформанси.

Датасет
Овој проект не користи класичен датасет од Kaggle.

Извор на податоци: Јавно достапни информации од онлајн извори како Minecraft Wiki, Fortnite Wiki и други гејминг ресурси.

Големина и формат: Нема локално зачуван датасет и не се симнуваат податоци.

Како ќе се користи: Информациите се користат како референца преку AI моделот. Не се користи web scraping и не се складираат веб-страници.

Карактеристики на проектот
- Интерактивен AI chatbot за гејмери со модерен веб-интерфејс
- Дава совети, објаснувања и tips & tricks
- Одговара на прашања во реално време
- Работи локално преку Flask веб-сервер
- Модерен responsive дизајн кој работи на сите уреди
- Пример прашања за брз почеток
- Typing индикатор додека AI размислува
- Поддршка за разговор во реално време

Технологии користени во проектот
- Python 3.x – Програмски јазик
- Flask – Веб-фрејмворк за серверската логика
- Google Gemini API – AI модел за генерирање одговори
- HTML/CSS/JavaScript – Кориснички интерфејс
- python-dotenv – За управување со API клучеви

Поставување на проектот
1. Клонирање на репозиториумот
2. Инсталирање на зависностите: pip install -r requirements.txt
3. Додавање на Gemini API клуч во .env фајлот
4. Стартување на апликацијата: python main.py
5. Отворање на http://127.0.0.1:5000 во веб-прелистувач

Очекуван резултат
Конечниот резултат е функционален AI Gaming Assistant со модерен веб-интерфејс кој работи локално преку PyCharm. Корисникот може да поставува прашања преку веб-интерфејс, а системот враќа текстуални одговори во реално време. Проектот е поставен во GitHub репозиториум заедно со оваа документација, source код, и примери на прашања и одговори.
